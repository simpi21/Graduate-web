Getting started

Before getting a start and work with Educati you need to know about a few important things. we assume that you have a basic knowledge about HTML/CSS/JS and SCSS ( optional but recommended :) ). we will work you through on every single detail

Dev environment

Please Note: Educati is built using most modern development toolset. we also focus on simple and easy dev environment setup but keep in mind dev environment setup only for the advanced user who wanna speed up development process with professional tools. this step is absolutely optional for the end user if you have basic knowledge about HTML/CSS/JS you can highly customize Educati. we will find details documentation how you can set up your dev environment at the end of this section

Component Based Development (CBD)

Everything in Educati is a component. components let you split the UI into independent, reusable pieces, and think about each piece in isolation.

Utility First CSS

Every components CSS written in Functional CSS (FCSS) concept come from functional programming. Small, clear, easy to read classes that are easy to apply and do one thing. Which, in essence, brings styling to the HTML level rather than the CSS level. For more details please read

  • Functional Programming, CSS, and your sanity
  • Functional CSS (FCSS)
  • Immutable CSS

Dynamic Scripts Loading

Dynamic Scripts Loading mean only load scripts when necessary. it will render page without blocking and load side 4X faster. More

Project Structure

Before talk about code principle let's walk through on project structure. 

        Educati
        ├── dist/
        │   ├─assets/
        │   ├─├─ css/
        │   ├─├──├─ style.css - (contain all theme style css)
        │   ├─│  ├─ vendors.bundle.css - (all third party vendors css include bootstrap. Note: Bundle using glup)
        │   ├─│  ├─ customize.css - (Your all custom css code)
        │   ├─├─ data/ - (Contain all json files)
        │   ├─├─ fonts/ - (Contain icon fonts)
        │   ├─├─ img/ - (Contain All images)
        │   ├─├─ js/ 
        │   ├─├──├─ scripts.js - (contain theme scripts)
        │   ├─│  ├─ vendors.bundle.js - (Scripts that are used in every page like jquery, bootstrap, wow)
        │   ├─├─ media/  - (Video, Audio, or any media)
        │   ├─├─ scss/  - (All scss files with partials)
        │   ├─├─ vendor/ - (All third party scripts and files)
        │   ├─blog-{blog-page-name}.html (All blog pages start with blog prefix)
        │   ├─element-{element-name}.html (All element prefix with element)
        │   ├─page-{page-name}.html (All pages prefix with page)
        │   ├─shop-{shop-page-name}.html (All shop pages start with shop prefix)
        ├── documentation - contain documentation of the theme
        ├── .bowerrc ( bower config file)
        ├── bower.json (Kepp track all third party plugin and it's dependency)
        └── gulpfile.js (gulp task runner)
        └── package.json (Keep track all npm packages used in project)

HTML files

Educati HTML written in most semantic way to improved seo and easy customization. HTML follow BEM naming convension

Educati has highly written in advance SCSS which compile to plain CSS so that browser can understand it. to compile SCSS you can use gulp task runner or very popular software like Prepros. if you don't like command line interface you can very easily use Prepose no configuration required just drop your project folder into Prepose and you good to go. Setup development environment (Optional)

Install Node.js Official site

Is a JavaScript runtime built on Chrome's V8 JavaScript engine. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient. Node.js' package ecosystem, npm, is the largest ecosystem of open source libraries in the world.

Install NPM Official Site

npm makes it easy for JavaScript developers to share and reuse code, and makes it easy to update the code that you’re sharing, so you can build amazing things.

npm is distributed with Node.js- which means that when you download Node.js, you automatically get npm installed on your computer.

Check that you have node and npm installed

To check if you have Node.js installed, run this command in your terminal:

node -v

To confirm that you have npm installed you can run this command in your terminal:

npm -v

Install GulpOfficial site

Gulp is a toolkit that helps you automate your time-consuming tasks in development workflow. To install gulp globally.

Installing packages necessary for Educati development

npm install

This will start the process of installing all dependencies which is listed inside package.json file.

Utility CSS

As refer before, Educati developed using utility classes. we assume you read reference article from Utility First CSS section and now have solid understand about Functional CSS. Please Note: Educati also heavily depends on Bootstrap utility classes specially flexbox. If you working with SCSS you will see all utility classes formatted nicely in utility folder.

let's walk through on utility class

Background

Class Properties
.bg-cover background-size: cover !important;
.bg-center background-position: center center !important;
.bg-absolute Set background as absolute position to any block

Border

Class Properties
.border-dashed border-style: dashed !important;
.border-dotted border-style: dotted !important;
.border-{md/lg}-{top/bottom/left/right} Add border in a specific media and side
.border-{1-6} border-width-{1-6}
.border-light{1-3} light border color variation
.border-white-0_{1-9} White border color variation
.border-black-0_{1-9} Black border color variation

Border radius

Class Properties
.rounded-{2-10/15-30} border-radius-{2-10/15-30}
.rounded-{top/bottom/left/right}-0 Reset border radius to 0
.rounded-{left/right} Pill border left/right side

Bottom/Top/Left/Right

Class Properties
.{bottom/top/left/right}-{0-50} .{bottom/top/left/right}-{0,5,10,15,20....50}
.{bottom/top/left/right}-{25p/50p/75p/100p} .{bottom/top/left/right}-{25/50/75/100}%

Center

Class Properties
.absolute-center Center absolute position element
.absolute-center-{x/y} Center absolute position element {x/y} axis

Flex

Class Properties
.flex-child-samewidth Make flex child same width
.flex-auto .flex-auto

Font Size

Class Properties
.font-size-{md/lg}-{10-60} Set font size 10,12,14,16,18...60. md/lg set specific media

Font Weight

Class Properties
.font-weight-{thin/semiBold/bold/extraBold} Set font weight

Height

Class Properties
.height-{50/60/70/80/90/100}vh height: {50/60/70/80/90/100}vh
.height-{1-15} height: {1-15}px
.height-100p height: 100%

Letter Spacing

Class Properties
.letter-spacing-{1-10} letter-spacing:{1-10}px

Line Height

Class Properties
.line-height-{reset/xs/sm/lg/xl} line-height:{normal/1.2/1.47/2/2.2}

List Style

Class Properties
.list-style-{circle/square} list-style-type: {circle/square};

Spacing

Class Properties
.{margin/padding} Please check details at scss/utilities/_{margin/padding}.scss

Opacity

Class Properties
.opacity-{01-09} opacity:{0.1-0.9}

Z-index

Class Properties
.z-index-{1,5,10,20,50,100} z-index:{1,5,10,20,50,100}

Let's walk through on third party components and there options

Carousels

Edicati used most popular Owl Carousels 2. as Educati used dynamic scripts loading .owl-carousel class is the hook to init carousel. you can pass rich option by HTML5 data- attribute. options

Option Default Description
data-items 3 The number of items you want to see on the screen.
data-items-tablet 2 The number of items you want to see on tablet pc .
data-items-mobile-wide 1 The number of items you want to see on wide mobile.
data-autoplay true Will slide auto.
data-autospeed 3000 Time to change slide one slide to another.
data-smartspeed 800 Time duration of change slide.
data-autoheight true Slider will adjust the height.
data-space 0 Space between slide.
data-dots false Show dots navigation.
data-dots-class false Class to add dots navigation.
data-arrow false Show next/prev navigation.

Counters

Educati used countTo plugin with apear js. you can find details Here

.ec-counter is the hook for load counter script

Option Default Description
data-to 100 he number to stop counting at. (default: 100)

Gallery

Lightbox
